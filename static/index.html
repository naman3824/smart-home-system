Scheduled guest access
Add recurring visitors — maid, cook, delivery agent, etc. — with specific allowed days and time windows. Visits outside the schedule are flagged as restricted access even if the face is recognized.
👥 Scheduled visitors
Loading…
📋 How it works
✅Within schedule: face detected, name matches a scheduled visitor, current time and day are within their window → logged as "authorized guest"
⚠️Outside schedule: face matches but wrong time or day → logged as "restricted access" with a warning — not a full intruder alert, but flagged for the owner
🚨Not in schedule: face not recognized and no schedule matches → full intruder alert (same as before)
To test: use the Simulations tab → Security events → "Unknown 🚨" button, then add that name here as a scheduled visitor and test again.
